# Smart Canteen Ordering System

## Project Overview
The **Smart Canteen Ordering System** is a web-based application designed to streamline ordering in college or office canteens. It eliminates long queues, reduces waiting times, and provides a smooth digital experience for both customers and canteen staff.

---

## Features

### User Features
- **Digital Menu:** Browse items with descriptions and prices.
- **Online Ordering:** Place orders directly from your device.
- **Order Status Tracking:** Check the status of your order in real-time.
- **Authentication:** Secure login, sign up, and forgot password features.

### Admin Features
- **Order Management:** View, update, and manage orders efficiently.
- **Menu Management:** Add, edit, or remove menu items.

---

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** Firebase Auth (Email/Password & Google Sign-In)
